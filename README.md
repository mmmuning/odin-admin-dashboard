# Admin Dashboard

This project is part of **The Odin Project’s Intermediate HTML and CSS Course**.
It’s a responsive admin dashboard layout built using **HTML**, **CSS Grid**, and **Flexbox**, designed to practice modern layout techniques and reinforce front-end structure and organization.

---

### This project includes:

- A complete dashboard interface with a sidebar, header, and main content area
- Responsive layout using **CSS Grid** and **Flexbox**
- Sections for **projects**, **announcements**, and **trending users**, filled with **dummy placeholder content**
- Semantic and organized HTML structure for readability and accessibility

---

### Through this project, I practiced and demonstrated:

- Building **complex, multi-section layouts** using **CSS Grid** and **Flexbox**
- Managing **responsive grid templates** and spacing for consistent alignment
- Using **custom properties (CSS variables)** for color and theme control
- Structuring reusable UI components such as cards and side menus
- Applying **box shadows, border-radius, and color hierarchy** for a modern, cohesive design
